name: explain_code
description: Explain complex code logic to the user.
---

**Steps:**
1. **High-Level Summary:** What does this module/function do in one sentence?
2. **Input/Output:** Describe data flow.
3. **Key Logic:** Walk through the algorithm step-by-step (using pseudocode if helpful).
4. **Why:** Explain *why* it was written this way (historical context or constraints, if known).
5. **Complexity:** Time/Space complexity analysis.
