---
name: generate_tests
description: Create comprehensive test suites for existing code.
---

**Steps:**
1. **Analyze Logic:** Identify branches, loops, and edge cases (nulls, empty lists, boundary values).
2. **Happy Path:** Write tests for the expected success scenarios.
3. **Edge Cases:** Write tests for boundary conditions.
4. **Error Handling:** Write tests to ensure errors are raised/handled correctly.
5. **Mocking:** Identify external dependencies and create Mocks/Stubs.
6. **Output:** A self-contained test file ready to run.
